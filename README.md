erklärung zu der Titel animation: 
1. dem Titel eine CSS‑Klasse (title-pulse) geben.
2. Die Klasse "animieren":
• 	pulse ist der Name der Animation
• 	1.8s ist die Dauer
• 	infinite bedeutet, sie läuft dauerhaft
• 	ease-in-out sorgt für weiche Übergänge
3. Im  @keyframes wird die Bewegung definiert:
• 	Start: normale Größe
• 	Mitte: etwas größer
• 	Ende: wieder normal
Dadurch entsteht ein sanfter „Beat‑Effekt“
